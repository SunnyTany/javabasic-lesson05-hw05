# Розрахунок суми податку

В класі **Main** реалізован функціонал розрахунку суми податку.

Розраховується сума податку від певної суми доходу за допомогою ***calculateTax***. 
Сума податку дорівнює відсотку від суми доходу.
Доходи оподатковуются за прогресивною шкалою оподаткування:

+ від 0 до 10000, включно, за ставкою 2,5%.
+ від 10000, виключно, до 25000, включно, за ставкою 4,3%.
+ більше 25000, за ставкою 6,7%, 
+ якщо від'ємне число, податок не розраховується.

Форматування податку до виду два знаки після десяткового роздільника призводиться за допомогою ***decimalFormatTax***.
